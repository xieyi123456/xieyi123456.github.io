Netty-优点，应用场景

netty 优点

异步的（采用多线程实现了调用方法与返回结果的分离）
基于事件驱动（本质上基于 IO 多路复用）

优势：

基于 NIO，但是改善了 NIO
1，需要自己构建协议
2，需要解决 TCP 传输问题，粘包半包。
3，NIO 中有 epoll 空轮询 bug。
4，对 API 增强了。bytebuff。

现在是 4.0
5.0 已经废弃，引入 AIO，性能未提升。
